## Core Features:

- Product Listing: Allow the admin to add new products to the inventory, including details like name, description, purchase cost, sale price, and quantity.
- Inventory Tracking: Real-time tracking of product quantities as stock enters and leaves the inventory, using tools in the LLM to suggest reorder points.
- Sales Logging: Record each sale, automatically updating the inventory to reflect the sold quantity.
- Reporting: Generate reports showing daily, weekly, monthly, and custom date range sales, inventory levels, and profit margins. Includes exportacion de reportes excel, hmtl
- Admin Account: Secure single admin account to manage the store inventory.
- Inflow and Outflow Management: Registering Inflow and Outflow operations.

## Style Guidelines:

- Primary color: Dusty Rose (#C4A4A4) for a soft, reliable, and classic aesthetic.
- Background color: Beige (#F5F5DC), subtly similar to the primary hue, but much lighter for a gentle contrast.
- Accent color: Pale Olive (#BDB76B), providing an analogous contrast to the dusty rose and adding a touch of sophistication.
- Body and headline font: 'Literata', serif for an accessible vintage feel.
- Use minimalist, line-based icons to represent product categories and actions.
- Clean, card-based layout for easy readability and quick information access.
- Subtle transitions and animations to provide feedback on user actions.
